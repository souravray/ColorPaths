ColorPath
=========
משחק זה הותאם במיוחד ללקוי ראייה המתקשים לשחק במשחק יומיומי שלנו נראה קל ופשוט לראייה. 
התאמנו את המשחק בצבעיו, צורתיו, גודלו ליכולותם של לקויי הראייה לשחק משחק בדיוק כמו שאנו משחקים, רק בתנאים מותאמים אליהם (: 
An opensource HTML puzzle game inspired by Flow (http://itunes.apple.com/us/app/flow-free/id526641427?mt=8)
